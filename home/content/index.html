# 关于目录页面

目录页面是一个.md文件，可以存放在三级目录的任一级中，它的 font matter 中需要包含几个必须的要素：

pageComponent: 
  name: Catalogue
  data: 
    key: foldername
    imgUrl: /img/web.png
    description: 填写目录内容的介绍

# 关于侧边栏

sidebar: true
1

代表开启侧边栏，改成false就不会有了。

侧边栏是根据文件名来生成的，嵌套则是根据文件夹结构来进行，也许有根据文件二级标题形式进行配置的形式，不过暂时不是重点。
